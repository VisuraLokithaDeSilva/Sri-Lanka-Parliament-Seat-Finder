# 🏛️ Sri Lanka Parliament Seat Finder

![Project Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Tech Stack](https://img.shields.io/badge/Tech-HTML%20%7C%20CSS%20%7C%20JS-blue)

A responsive web application designed to help users locate the seating positions of Members of Parliament (MPs) in the Sri Lankan Parliament. The application features an interactive visual layout of the parliament chamber and a dual-search functionality.

## 🌟 Features

* **Interactive Seating Map:** A visual representation of the parliament floor plan. Clicking on a seat highlights it and displays the assigned MP's name.
* **Smart Search:**
    * **By Name:** Auto-suggests MPs as you type.
    * **By Seat Number:** Locate specific seats (e.g., "G-01", "O-10").
* **Color-Coded Layout:**
    * 🔴 **Red (G/OG):** Government/Ruling Party seats.
    * 🔵 **Blue (O):** Opposition seats.
    * 🟣 **Purple:** Cross-bench/Specific designated seats.
* **Mobile Responsive:** Fully optimized interface for both desktop and mobile devices.
* **External Integrations:** Links to the *Member Directory App* and *Staff Point of View* version.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, Flexbox for the seating grid, and responsive media queries.
* **JavaScript (Vanilla):** DOM manipulation, search logic, and dynamic grid generation.
* **Analytics:** Integrated with Google Analytics (GA4) and Microsoft Clarity for user behavior tracking.
